# Road Accident Prediction System

A machine learning-based web application for predicting road accidents using various traffic and environmental factors.

## Features
- Predict road accident probability based on multiple factors
- Interactive web interface
- Data visualization and analysis tools
- Upload and preview datasets

## Deployment
This application is deployed on Render and can be accessed at your Render app URL.

## Local Development
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Access at `http://localhost:5000`

## Technology Stack
- Python Flask
- Scikit-learn
- Pandas
- HTML/CSS/JavaScript
- Bootstrap

## Model
The application uses a trained machine learning model to predict road accidents based on:
- Location (State/UT)
- Junction type
- Vehicle age
- Human demographics
- Safety precautions
- Area type
- Traffic conditions
- Weather conditions
- Road type
- License type
- Time of day
